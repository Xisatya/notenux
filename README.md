# Notenux
**Notenux** is a lightweight text editor for **Linux**, built with **C++**.  
The goal is to provide a simple and clean alternative to WordPad-like editors on Linux, while staying minimal and efficient.

---

## Features
- [ ] Create a new text file  
- [ ] Open `.txt` files  
- [ ] Save files  
- [ ] Undo / Redo  
- [ ] Basic RTF/Markdown support  

---

## Build & Run
Make sure you have a C++ compiler installed (e.g., `g++`).

```bash
g++ src/main.cpp -o notenux
./notenux
